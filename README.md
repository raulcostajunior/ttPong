# ttPong
ttPong, two thumbs Pong, is the traditional Pong revisited for mobile phones.

It uses Sprite-Kit and has some pretty basic integration with GameCenter.
As of October, 2020, ttPong is available at the AppStore as a free application.

This is actually the first game I ever wrote. I followed a really simple and
smart advise I read somewhere for starting simple (like a Pong clone) instead of 
frustating myself by pursuing something more advanced that would, almost certainly,
not be completed.
