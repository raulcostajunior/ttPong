# ttPong
ttPong, **t**wo **t**humbs **Pong**, is a revisit to the traditional Pong game for mobile
phones. The basic idea is to be your own "opponent" and to win by hitting the
disc (ball) as many times as you can. This differs from the original game where
each player had their own score and the goal was to win the match.

ttPong is a native iOS app written in Swift. It uses Sprite-Kit and has some
pretty basic integration with GameCenter for supporting global Leader Boards.
As of October, 2020, ttPong can installed from the *AppStore* as a free application.

This is actually the first game I ever wrote. I followed a really good and
smart advise I read somewhere for starting simple instead of getting frustrated by 
pursuing something more ambitious that I would, almost certainly, not complete.
